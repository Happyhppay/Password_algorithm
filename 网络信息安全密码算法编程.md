# 网络信息安全密码算法编程作业要求及评分细则

**学校**：北京邮电大学

## 一、技术范围

需编程实现以下密码算法：

1. **对称加密算法**：AES、SM4、RC6

2. **哈希算法**：SHA1、SHA256、SHA3、RIPEMD160；HMacSHA1、HmacSHA256、PBKDF2

3. **编码算法**：Base64、UTF\-8 编码

4. **公钥密码算法**：RSA\-1024bit、ECC\-160bit、RSA\-SHA1、ECDSA

## 二、技术要求

2. 支持第三方程序通过**接口调用**，并返回结果与执行状态代码。
